## 🚀 Quick Start

### Online (Instant)
Visit: **[kl4rkx.github.io/free-iptv-player](https://kl4rkx.github.io/free-iptv-player)**

### Local Development
```bash
git clone https://github.com/Kl4rkx/free-iptv-player.git
cd free-iptv-player
python -m http.server 8080
```
Open http://localhost:8080
